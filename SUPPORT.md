# Support

## How to get help
- Usage/documentation questions: open a GitHub Discussion or Issue.
- Bug reports: open a Bug Report issue using the provided template.
- Security concerns: follow `SECURITY.md` and use private reporting.

## Before opening an issue
- Check `README.md`
- Check `/howto`
- Check `/howto/troubleshooting.md`
- Run `theme-extractor doctor` and include output when relevant

## What to include
- command used
- relevant flags/env
- minimal reproduction steps
- expected vs actual behavior
- logs/error messages
